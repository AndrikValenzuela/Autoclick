
1. Al ejecutar, el programa te pedirá:
   - Tecla para CLICK IZQUIERDO (presiona la que quieras, ej: Z)
   - Tecla para CLICK DERECHO (presiona la que quieras, ej: X)
   - Velocidad de clicks (1-50 por segundo, default: 10)

2. Uso:
   - MANTÉN PRESIONADA la tecla = clickea continuamente
   - SUELTA la tecla = para de clickear
   - Cierra la ventana para salir del programa
